# Bienvenido a Escanor 👋

Este es un proyecto desarrollado con [Expo](https://expo.dev) creado con [`create-expo-app`](https://www.npmjs.com/package/create-expo-app).

## Acerca de

**Escanor** es una aplicación que ayuda en la verificación legal de los billetes de Bs.10, Bs. 20 y Bs. 50 de la Serie B, según la [normativa vigente](https://abi.bo/guia-que-hacer-si-tiene-billetes-de-bs-10-20-o-50-de-la-serie-b/) en Bolivia.
